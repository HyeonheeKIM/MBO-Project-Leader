Features

핵심 기능

MBO 목표 관리에 필요한 모든 기능을 하나의 시스템에서 제공합니다.

📅

연도별 MBO 관리

연도 관리 페이지에서 연도를 추가/선택합니다. 각 연도별로 독립적인 프로젝트 목표를 관리하며, 과거 데이터를 참조할 수 있습니다.

📂

프로젝트별 목표 설정

프로젝트명, 설명, KPI, 가중치(%), 우선순위(⭐1~5), 상태(대기/진행중/완료/취소)를 설정합니다. 수정·삭제가 자유롭습니다.

📆

월별 계획 수립

12개월 카드 뷰에서 각 월의 마일스톤과 목표를 설정합니다. 월별로 상태(미완료/진행중/완료)와 비고를 기록할 수 있습니다.

✅

일별 실행 & 체크

달력 UI에서 날짜를 선택하고, 태스크를 추가합니다. 체크박스로 완료 여부를 토글하면 진행률이 자동 계산됩니다.

📈

추적 & 분석

프로젝트별 태스크 달성률, 월별 달성률, 가중치, 완료 태스크/마일스톤 수를 한눈에 확인합니다. 12개월 인디케이터로 진행 상태를 시각화합니다.

📊

대시보드 & 통계

전체 프로젝트 수, 완료 수, 진행중 수, 평균 달성률, 오늘 태스크 현황 등 핵심 통계를 대시보드에서 실시간으로 확인합니다.

Workflow

사용 흐름

4단계의 간단한 흐름으로 체계적인 MBO 관리를 시작하세요.

1

연도 · 프로젝트 등록

[연도 관리]에서 연도를 추가하고, [프로젝트] 메뉴에서 KPI·가중치·우선순위를 설정합니다.

2

월별 계획 설정

[일별 체크]에서 달력으로 날짜 선택 후 태스크를 추가하고, 체크박스로 완료 처리합니다.

[대시보드]와 [추적/분석]에서 달성률·통계를 확인하고 목표를 조정합니다.

Database

데이터 구조

SQLite 4개 테이블로 모든 데이터를 체계적으로 관리합니다.

📆

years

year (PK) — 관리 대상 연도
여러 연도를 등록하고 연도별로 전환합니다.

📂

projects

id, year, name, description, kpi, weight, priority, status, created_at
연도별 프로젝트를 저장합니다.

📅

monthly_plans

id, project_id, month, milestone, target, status, note
프로젝트별 12개월 마일스톤을 관리합니다.

✅

daily_tasks

id, project_id, task_date, title, description, is_done, priority, created_at
일별 태스크를 기록합니다.

Pages

6개 화면 상세 안내

사이드바에서 각 화면으로 이동합니다.

📋 대시보드

선택 연도의 전체 통계(프로젝트 수, 완료 수, 진행중, 평균 달성률, 오늘 태스크)를 카드로 표시합니다. 하단에 프로젝트 목록 테이블이 상태·진행률 바와 함께 나타납니다.

📂 프로젝트

프로젝트를 추가·수정·삭제합니다. 각 카드에 프로젝트명, KPI, 가중치(%), 우선순위(⭐), 상태, 진행률 바가 표시됩니다. 삭제 시 관련 월별 계획·일별 태스크도 함께 삭제됩니다.

📅 월별 계획

프로젝트를 탭으로 선택 후, 1~12월 카드 그리드가 나타납니다. 각 카드를 클릭하면 마일스톤·목표·상태(미완료/진행중/완료)·비고를 입력하는 대화상자가 열립니다.

✅ 일별 체크

좌측: 달력(◀▶ 월 이동, 날짜 클릭 선택, 태스크 있는 날에 · 표시). 우측: 해당 날짜의 태스크 목록. 체크박스로 완료 토글, 🗑️로 삭제. [+ 태스크 추가]로 제목·설명·날짜·우선순위를 입력합니다.

📈 추적/분석

프로젝트별로 태스크 달성률, 월별 달성률, 가중치, 완료 태스크 수, 완료 마일스톤 수를 통계 카드로 표시합니다. 전체 진행률 바와 12개월 상태 인디케이터가 포함됩니다.

📆 연도 관리

[+ 연도 추가]로 2000~2100 사이 연도를 등록합니다. 각 연도 카드에 프로젝트 수가 표시되며, 클릭하면 해당 연도 대시보드로 이동합니다.

Usage Guide

사용 방법 안내

프로그램 실행부터 프로젝트 추가·관리까지 전체 흐름을 단계별로 안내합니다.

0

프로그램 실행

python Project_Leader.py

터미널(CMD / PowerShell)에서 Project_Leader.py가 있는 폴더로 이동 후 위 명령을 실행합니다.
처음 실행하면 mbo_project_leader.db 파일이 자동 생성되고, 현재 연도(2026)가 기본 등록됩니다.
추가 패키지 설치는 필요 없습니다 — Python 표준 라이브러리(tkinter, sqlite3)만 사용합니다.

1

연도 확인 · 추가

사이드바 → 📆 연도 관리

확인할 것

프로그램을 처음 실행하면 현재 연도(2026)가 자동으로 등록되어 있습니다. 다른 연도를 관리하고 싶으면 이 화면에서 추가합니다.

연도 추가 방법

우측 상단 [+ 연도 추가] 클릭 → 연도 입력 (예: 2027) → 확인. 등록된 연도 카드를 클릭하면 해당 연도 대시보드로 이동합니다.

💡 팁: 모든 다른 화면 상단에도 연도 드롭다운이 있어서, 언제든 연도를 전환할 수 있습니다.

2

프로젝트 추가 ⭐

사이드바 → 📂 프로젝트

프로젝트 화면에서 우측 상단 [+ 프로젝트 추가] 버튼을 클릭하면 입력 대화상자가 열립니다.

📝 입력 필드 설명

프로젝트명 *
프로젝트 이름 (필수 입력). 예: "AI 챗봇 개발", "ERP 고도화"
설명
프로젝트에 대한 상세 메모 (여러 줄 입력 가능, 선택)
KPI
핵심 성과 지표. 예: "고객 만족도 90% 달성", "월 매출 1억"
가중치 (%)
연간 MBO에서 이 프로젝트의 비중. 숫자 입력 (예: 30)
우선순위
드롭다운에서 1~5 선택. 높을수록 목록 상단에 표시됨 (⭐ 수로 표현)
상태
드롭다운: 대기 · 진행중 · 완료 · 취소 중 선택

✅ 저장

모든 항목을 입력한 후 [저장] 버튼 클릭. 프로젝트 목록에 즉시 반영됩니다.

✏️ 수정 · 🗑️ 삭제

프로젝트 카드 우측 상단의 [✏️ 수정]으로 수정, [🗑️ 삭제]로 삭제. 삭제 시 월별 계획·일별 태스크도 같이 삭제됩니다.

3

월별 계획 설정

사이드바 → 📅 월별 계획
  1. 상단에서 프로젝트 탭을 클릭하여 대상 프로젝트를 선택합니다.
  2. 1월 ~ 12월 카드 그리드가 나타납니다. 설정하려는 월 카드를 클릭합니다.
  3. 대화상자에서 다음 항목을 입력합니다:
    마일스톤 — 해당 월의 핵심 산출물 (예: "프로토타입 완성")
    목표 — 달성할 목표 (예: "기능 테스트 완료")
    상태 — 미완료 / 진행중 / 완료 선택
    비고 — 추가 메모
  4. [저장] 을 누르면 카드에 마일스톤·목표·상태가 표시됩니다.

💡 팁: 같은 카드를 다시 클릭하면 기존 내용을 수정할 수 있습니다.

4

일별 태스크 추가 · 체크

사이드바 → ✅ 일별 체크

📅 달력 (좌측)

  • ◀ / ▶로 월 이동
  • 날짜를 클릭하면 파란 배경으로 선택됨
  • 오늘 날짜는 청록색으로 표시
  • 태스크가 있는 날은 ·숫자로 표시
  • 일요일은 빨간색

📝 태스크 (우측)

  • [+ 태스크 추가] 클릭 → 대화상자
  • 입력: 제목(필수), 설명, 날짜, 우선순위(1~5)
  • 체크박스 클릭 → 완료 처리 (취소선 표시)
  • 🗑️ 클릭 → 확인 후 삭제
  • 완료 상태는 진행률에 자동 반영

💡 팁: 프로젝트 탭을 먼저 선택해야 합니다. 태스크는 선택된 프로젝트에 속합니다.

5

대시보드 · 추적/분석 확인

📋 대시보드

  • 전체 프로젝트 수 · 완료 · 진행중
  • 평균 달성률 (자동 계산)
  • 오늘 태스크 완료/전체
  • 프로젝트별 상태·진행률 테이블

📈 추적/분석

  • 프로젝트별 태스크 · 월별 달성률
  • 가중치 · 완료 태스크 · 마일스톤 수
  • 전체 진행률 바
  • 12개월 상태 인디케이터 (색상)

💡 달성률은 일별 태스크 완료 수 / 전체 태스크 수 × 100으로 자동 계산됩니다.

📌 전체 흐름 요약

프로그램 실행 → 연도 확인 → 프로젝트 추가 → 월별 계획 → 일별 태스크 → 대시보드 확인

이 순서대로 진행하면 MBO 목표를 체계적으로 관리할 수 있습니다.
